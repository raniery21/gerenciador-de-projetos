GitHub

Crie um Repositório no GitHub para o seu projeto, alguns comandos importantes:

git init
git clone (URL do Repositório)
git add --all
git commit -m ("Nome do seu commit")
git push origin master

GitHub Issues

Você pode criar uma lista de tarefas na parte "Issues" do GitHub utilizando (- [ ] "Nome da tarefa")
